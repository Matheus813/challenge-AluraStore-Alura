Análise de Performance: Alura Store Brasil
Descrição do Projeto
Este projeto consiste na análise exploratória de dados de vendas de quatro lojas (Loja 1 a Loja 4) da Alura Store Brasil, utilizando um notebook Python. O objetivo principal é avaliar o desempenho de cada unidade em termos de receita (Faturamento) e custos operacionais (Frete), fornecendo insights valiosos para a tomada de decisão.

Objetivo Principal
Identificar as lojas com melhor e pior performance de Lucratividade e entender os fatores (como faturamento e frete médio) que influenciam esses resultados.

Tecnologias e Ferramentas
O projeto foi desenvolvido em Python, usando as seguintes bibliotecas e ambientes (perfeito para quem está mergulhando em IA e Data Science!):

Linguagem: Python

Principais Descobertas (Insights)
A análise focada nos gráficos gerados no notebook AluraStoreBrasil.ipynb revelou o seguinte:

---A Loja Menos Lucrativa é a Loja 4---
A Loja 4 demonstrou ser a unidade menos lucrativa, e a causa principal reside no baixo volume de vendas:
Faturamento Total (Receita): A Loja 4 registrou o menor faturamento total entre todas as unidades, sendo o fator decisivo para sua baixa lucratividade.
Frete Médio (Custo): Embora a Loja 4 tenha a vantagem de possuir o menor frete médio (23.6%) — o que ajudaria a margem de lucro —, essa vantagem não é suficiente para compensar a falta de receita.
